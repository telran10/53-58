                           

1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?


2. Как получить на локальный комьютер изменения с github.com

3. Какую информацию нам дают команда :
   git status
   git log
   git remote -v 


 1. 
    1. Заходим или создаем папку в которую будем копировать репазиторий.
       Наводим мышку на эту папку и с контегстного меню заходим в gitbash.
       Командой git config --global -l проверяем user.name и user.email/
       и проверить связано ли эта папка с git_ом, должна быть полупрозрачная 
       папочка git

    2. Теперь зоходим в githab на аватарке нажимаем New repositories и 
       выбираем из списка нужный. Затем нажимаем зеленую кнопочку code 
       выбираем ssh, копируем адресс.
    3. Заходим в gitbash пишем команду git clone и через пробел вставляем
       скапированный адресс. Поле ввода появится сообщение из 6 строк в 
       большинстве из них будет написано 100% 
    4. Можна проверить камандой ls

 2. При вводе команды git clone  мы получаем копию репозитория 
            а команда git fetch для получения новых работ создаными другими людьми

 3. git status        показывает на какой ветке и в каком состоянии или статусе обнаруженные файлф
    git log           показывает каммиты в обратном порядке
    git remote -v     команда проверяет связь репозитория на ПК с githab_oм
